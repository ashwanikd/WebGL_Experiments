Depth First Search Algorithm for Maze Generation

Interactive Graphics Implementation

Algorithm

Depth First Search(DFS) Algorithm is a searching algorithm in graph theory where we are given a graph G(V,E) where V is the set of vertices(nodes) and E is the set of Edges connecting vertices, we are given with a start node and the node that we need to find in the graph i.e. the goal node. now the algorithm proceeds as follows.

  1. make all nodes unvisited
  2. push the first node onto the stack
  3. while the stack is not empty do
  4.  pop the first element from stack let it be v
  5.  if v is the goal node then return
  6.  make v as visited
  7.  push all nodes adjacent to v which are not visited on to the stack

Here is also a recursive implementation of the Algorithm which must be called by start node as the initial parameter. this one is more efficien and mostly used implementation of DFS.assume initally all nodes were unvisited.

Procedure DFS(node n)
  1. if n is goal node then return
  2. make n as visited node
  3. for each node i adjacent to n
  4.  if i is not visited then
  5.  DFS(i)

The way Algorithm proceeds will not make any loops in the path followed by the algorithm, by this strategy we will get a tree from the paths followed by the DFS algorithm.

Maze Generation

In our implementation we will have to traverse the whole graph starting from a random node and adding the edges that the Algorithm follows during the traversal of graph, which will give us the tree that we want and this will ultimately be the maze we wanted. also the choice of the vertext that the algorithm explores next in the algorithm step 3 of Procedure DFS will be taken random which will give us the complete maze.
